# Docker-k3d

Creates a lightweight, multi-node cluster running k3s over Docker

### Initializing cluster
``` bash
$ cd ./scripts
$ source ./init-cluster.sh
```

### Destroying cluster
``` bash
$ cd ./scripts
$ source ./destroy-cluster.sh
```
